# 介绍
**TORM** 为对象-关系映射(Object-Relational Mapping，简称ORM)框架 ，是【七天系列】中的其中一篇关于ORM框架的项目，写这项目的目的主要有以下几点：

- 用最少的代码来实现一款ORM框架
- 通过TORM进而理解ORM实现原理
- 去了解框架设计的奥妙
